This is a sample webpage demonstrating the use of semantic HTML5 tags such as <header>, <nav>, <main>, <aside>, <article>, <section>, and <footer>.

Semantic HTML5 tags provide meaning to the web content, allowing developers to create more organized and readable code. This enhances the overall user experience and aids in SEO efforts.
